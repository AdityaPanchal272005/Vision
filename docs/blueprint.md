# **App Name**: Visionary Vault

## Core Features:

- Interactive Hero Section: Hero section featuring a full-screen, dark 3D Spline camera with a static image fallback to enhance initial user engagement and provide graceful degradation for unsupported browsers.
- Featured Portfolio Strip: Display a curated selection of photographs from the portfolio in an easily consumable horizontal strip.
- Portfolio Showcase: Organize and present photography work in a dynamic, filterable layout with lightbox to deliver high-quality images. Includes filter categories All, Weddings, Pre-weddings, and Events
- Private Client Galleries: Enable clients to access hidden, password-protected photo galleries. This system uses unique UUIDs for secure sharing, and galleries will not be indexed by search engines.
- Online Booking System: Allow users to book photography services through a contact form that validates inputs and integrates a honeypot to filter out spam. Upon successful submission, the app sends the details to the photographer.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to bring a sense of creative professionalism to the portfolio, like that used by creative agencies.
- Background color: Dark Slate Gray (#2F4F4F), a dark shade from the same color family as Deep Indigo, chosen to set off and complement the photography showcased.
- Accent color: Violet (#8A2BE2), a vivid, bright shade, providing striking visual contrast to call attention to buttons and links.
- Headings: 'Belleza' sans-serif, for fashion and design-aligned style
- Body: 'Alegreya' serif, to use if longer text passages are added
- Simple, modern icons to represent services and navigation.
- Clean, grid-based layout with plenty of whitespace for a modern feel.
